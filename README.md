
# YouTube Channel Data Analysis Dashboard

## 📊 Project Overview
This project analyzes YouTube channel performance data and presents insights through an interactive dashboard. The goal is to understand trends in views, subscribers, engagement, and video performance using data analysis techniques.

The project demonstrates practical skills in **data cleaning, analysis, and visualization** using Python and dashboard tools.

---

## 🚀 Objectives
- Analyze YouTube channel performance metrics
- Identify trends in views, likes, and subscriber growth
- Build a clear and interactive dashboard for insights
- Practice real-world data analysis workflow

---

## 🛠️ Tools & Technologies Used
- **Python**
- **Pandas**
- **Excel**
- **Power BI**
- **Git**
- **GitHub**

---

## 📂 Project Workflow

1. **Data Collection**
   - Collected YouTube channel data from public sources / datasets.

2. **Data Cleaning**
   - Removed missing values
   - Standardized column formats
   - Prepared data for analysis.

3. **Data Analysis**
   - Analyzed views, likes, comments, and subscriber trends.
   - Identified high-performing videos.

4. **Data Visualization**
   - Created charts and graphs to highlight patterns.
   - Built a dashboard for better interpretation.

---

## 📈 Key Insights
- Videos with higher engagement tend to generate more views.
- Upload frequency impacts subscriber growth.
- Certain categories of videos perform better than others.
